# 🍔 Projeto Cantina Fatec 2026

Este projeto consiste em um sistema de gerenciamento para uma cantina universitária, desenvolvido com foco em **POO** e persistência de dados.

## 🏗️ Estrutura
* **main.py**: Ponto de entrada da aplicação.
* **cantinadados.py**: Módulo de manipulação e armazenamento.
* **/modelos**: Definições das classes.
* **/estruturas**: Estruturas de dados do inventário.

## 🛠️ Tecnologias
* **Python 3**
* **Pickle**: Serialização e persistência de dados.
* **Faker**: Geração de dados fictícios para testes.
